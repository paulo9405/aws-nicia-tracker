# Diagnóstico Diferencial — Síndrome Respiratória

> Cruza as doenças com sinais respiratórios já documentadas nos arquivos individuais. **As fontes
> são os arquivos de cada doença** (Merck/MSD/WOAH/PMC — ver cada arquivo citado). Ferramenta de
> orientação; confirmação sempre por laboratório.

## Pergunta central: quais sinais/lesões separam as respiratórias?

| Doença (arquivo) | Pistas que chamam atenção | Lesão que ajuda a distinguir | Confirmação |
|---|---|---|---|
| **Influenza Aviária HP** (`V01`) | Morte súbita, cianose de crista, quadro **sistêmico**, alta mortalidade | Necrose de crista/barbela, hemorragias sistêmicas, edema de tarsos | **RT-PCR** (notificar!) |
| **Newcastle velogênica** (`V02`) | Respiratório **+ neurológico + digestivo** (diarreia verde), queda de postura | Hemorragia na mucosa do **proventrículo**, botões necróticos intestinais | **RT-PCR** (notificar!) |
| **Bronquite Infecciosa** (`V04`) | Respiratório + **queda de postura/casca ruim** + (cepas renais) nefrite | Traqueíte/aerossaculite; rins inchados com uratos; oviduto ("falsa poedeira") | RT-PCR + **tipagem do gene S** |
| **Laringotraqueíte** (`V06`) | **Muco/sangue na traqueia**, tosse com expectoração sanguinolenta | **Traqueíte hemorrágica** com cilindros; inclusões intranucleares | PCR + histopatologia |
| **Metapneumovírus / SHS** (`V12`) | **Cabeça inchada**, seios inflados; torcicolo por otite | Edema subcutâneo de cabeça; sinusite | RT-PCR **precoce** |
| **Micoplasmose MG** (`B04`) | Respiratório **crônico**, espuma ocular, seios inchados; piora com estresse | **Aerossaculite** (espessamento de sacos aéreos) | Sorologia + PCR |
| **Coriza Infecciosa** (`B05`) | **Cara inchada** de início rápido, secreção purulenta, queda de consumo | Sinusite/edema facial | Cultura + PCR |
| **ORT** (`B07`) | Respiratório, frequentemente **secundário/coinfecção** | **Pneumonia** (muitas vezes unilateral), pericardite | Cultura + PCR |
| **Cólera Aviária** (`B06`) | Morte súbita em **adultos**, diarreia verde | Focos necróticos hepáticos + petéquias | Cultura (bipolares) |
| **Colibacilose** (`B03`) | Geralmente **secundária** a viral/ambiência | **Peri-hepatite + pericardite + aerossaculite** | Cultura de sítio estéril |
| **Aspergilose** (`F01`) | Dispneia **sem estertores altos**, pintinhos, história de mofo | **Nódulos branco-amarelados** em pulmão/sacos aéreos | Histopatologia/cultura (hifas) |
| **Bouba diftérica** (`V09`) | Placas na boca/laringe (forma úmida) | Membranas diftéricas; corpúsculos de Bollinger | Histopatologia/PCR |

## Regras de raciocínio
- **Quadro sistêmico + alta mortalidade + notificável** → priorizar **IA e Newcastle** e **notificar**.
- **Sangue na traqueia** → Laringotraqueíte.
- **Cabeça/cara inchada** → diferenciar **Coriza** (bacteriana, rápida) de **aMPV/SHS** (viral + secundárias).
- **Peri-hepatite/pericardite/aerossaculite** → colibacilose (buscar **causa primária**).
- Respiratórias frequentemente ocorrem em **complexo** (viral + micoplasma + *E. coli* + ambiência).

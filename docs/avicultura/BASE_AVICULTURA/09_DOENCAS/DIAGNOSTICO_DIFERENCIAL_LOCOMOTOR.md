# Diagnóstico Diferencial — Síndrome Locomotora (Claudicação/Artrite)

> Distingue causas **infecciosas, articulares e metabólicas/nutricionais**. Fontes = arquivos
> individuais + Módulo 4.

| Doença/condição (arquivo) | Sinal-chave | Aspecto que distingue | Confirmação |
|---|---|---|---|
| **Artrite viral (Reovírus)** (`V11`) | Jarrete/tendão inchado, 4–7 sem | Tenossinovite **não purulenta**; ruptura de gastrocnêmio | RT-PCR + descartar bactéria |
| **Artrite estafilocócica** (`B09`) | Articulação inchada, claudicação | **Exsudato purulento** (vs. viral) | Cultura |
| **Espondilite por *E. cecorum*** (`B09`) | Paralisia de membros posteriores (frangos pesados) | Lesão em **coluna torácica** | Cultura/PCR |
| **Micoplasma synoviae** (`B04`) | Sinovite, jarrete inchado | Exsudato sinovial; + respiratório | Sorologia/PCR |
| **Marek (neural)** (`V05`) | **Paralisia assimétrica** | **Nervo aumentado** (não é artrite) | Histopatologia |
| **Discondroplasia tibial** (`M02`) | Claudicação, arqueamento | Cunha de **cartilagem não mineralizada** na tíbia | Necropsia; dieta (eletrólitos/Ca:P) |
| **Raquitismo** (`M02` / Módulo 4) | Ossos moles, jovens | "Beading" costal; ossos flexíveis | Ca/P/vit. D; bioquímica |
| **Perose (Mn/colina/B)** (Módulo 4, `06/05`) | Jarrete deformado, tendão deslocado | **Inespecífica** — Mn, colina, niacina, biotina, folato, Zn | Dieta/histórico |
| **Hipocalcemia/fadiga** (`M02`) | Paralisia em poedeira de pico | Ossos frágeis | Responde a cálcio |

## Regras de raciocínio
- **Articulação com pus** → bacteriana (Staph); **sem pus** → viral (reovírus).
- **Nervo aumentado** → Marek (não confundir com artrite).
- **Deformidade óssea sem inflamação** → metabólico/nutricional (discondroplasia, raquitismo, perose).
- **Perose é inespecífica** — investigar dieta (ver Módulo 4, arquivo 11).
- Paralisia de membros posteriores em frango pesado → considerar ***E. cecorum*** (emergente).

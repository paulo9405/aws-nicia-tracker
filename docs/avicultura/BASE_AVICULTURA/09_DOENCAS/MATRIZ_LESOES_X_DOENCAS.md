# Matriz — Lesões × Doenças

> Parte da **lesão de necropsia** para as doenças a considerar, com observação distintiva. Fontes =
> arquivos individuais + Módulos 4 e 18. Confirmação por histopatologia/laboratório.

| Lesão (achado de necropsia) | Doenças a considerar (arquivo) | Observação distintiva |
|---|---|---|
| **Traqueíte hemorrágica com cilindros** | LTI (`V06`) | Inclusões intranucleares; PCR |
| **Aerossaculite** | MG (`B04`), ORT (`B07`), BI (`V04`), *E. coli* (`B03`) | *E. coli* soma peri-hepatite/pericardite |
| **Peri-hepatite + pericardite + aerossaculite** | Colibacilose (`B03`) | Buscar causa primária (viral/ambiência) |
| **Hemorragia na mucosa do proventrículo** | Newcastle velogênica (`V02`) | + botões intestinais; notificar |
| **Necrose de crista/barbela + hemorragias sistêmicas** | Influenza HP (`V01`) | Notificar; RT-PCR |
| **Nódulos pulmonares branco-amarelados** | Aspergilose (`F01`) | Hifas na histopatologia |
| **Bursa edemaciada/hemorrágica → atrófica** | Gumboro (`V03`) | + hemorragia muscular (vvIBDV) |
| **Medula óssea pálida + timo atrófico** | CAV (`V10`) | Anemia; PCR |
| **Aumento de nervos periféricos** | Marek (`V05`), Reticuloendoteliose (`V14`) | Marek: assimétrico + idade |
| **Tumor na bursa (nodular)** | Leucose Linfoide (`V13`) | >14 sem; bursa tumoral (≠ Marek) |
| **Fígado com focos necróticos puntiformes** | Cólera (`B06`) | Morte súbita em adultos |
| **Fígado com lesões "em alvo" + cecos** | Histomonose (`P02`) | Praticamente patognomônico (perus) |
| **Cores/núcleos cecais com sangue** | Coccidiose (E. tenella) (`P01`) | Raspado com oocistos |
| **Membrana diftérica no jejuno/íleo** | Enterite Necrótica (`B08`) | Intestino friável; checar coccidiose |
| **Saco vitelínico não absorvido + nódulos viscerais (pintinho)** | Pulorose (`B01`), *E. coli* (`B03`), Salmonelas (`B02`) | Cultura |
| **Folículos ovarianos hemorrágicos/caseosos** | Pulorose/Tifo (`B01`), Gallibacterium (`B09`) | Sorologia/cultura |
| **Salpingite (oviduto com massa caseosa)** | *E. coli* (`B03`), Gallibacterium (`B09`) | Cultura diferencia |
| **Rins inchados + uratos** | BI nefropatogênica (`V04`), gota (Módulo 2/desidratação) | Tipar IBV; história de água |
| **Fígado gorduroso friável + hemorragia** | Esteatose/FLHS (`M01`) | Poedeira obesa; ≠ aflatoxina |
| **Líquido abdominal + coração direito dilatado** | Ascite (`M01`) | Crescimento rápido/altitude/frio |
| **Cunha de cartilagem não mineralizada (tíbia)** | Discondroplasia (`M02`) | Eletrólitos/Ca:P |
| **Membranas diftéricas boca/laringe** | Bouba úmida (`V09`), candidíase (`F02`), def. vit. A | Bollinger (bouba); leveduras (candida) |

> ⚠️ Evitar chamar lesão de "típica" sem descrevê-la; **"patognomônico" só com base sólida**
> (ex.: lesões em alvo da histomonose). Correlacionar lesão + sinal + epidemiologia + laboratório.

## Fontes
Todos os arquivos de doença do Módulo 9 + Módulos 4 (nutrição) e 18 (patologia/necropsia).

# Diagnóstico Diferencial — Síndrome Neurológica

> Sinais nervosos (torcicolo, paralisia, tremor, ataxia) são **muito inespecíficos** isoladamente.
> Cruza infecciosas e **nutricionais** (Módulo 4). Fontes = arquivos individuais + Módulo 4.

| Doença/condição (arquivo) | Sinal-chave | Idade/contexto | Como distinguir |
|---|---|---|---|
| **Newcastle (neurotrópica)** (`V02`) | **Torcicolo, paralisia, andar em círculo, espasmos** | Qualquer; + respiratório/digestivo | RT-PCR (notificar); quadro sistêmico |
| **Marek** (`V05`) | **Paralisia assimétrica** (perna à frente, outra atrás) | **≥ 3 semanas** | **Nervos periféricos aumentados** + tumores; histopatologia |
| **Encefalomielite Aviária** (`V07`) | **Tremor de cabeça/pescoço** + ataxia | **Pintinhos 1–3 sem**; transmissão vertical | Histopatologia (encefalomielite não supurativa); sorologia de matrizes |
| **Metapneumovírus/SHS** (`V12`) | Torcicolo/opistótono por **otite** | + cabeça inchada | RT-PCR |
| **Influenza HP** (`V01`) | Sinais nervosos + sistêmico | Alta mortalidade | RT-PCR (notificar) |
| **Def. vitamina E** (Módulo 4, `05`) | **Encefalomalácia**: ataxia, cabeça retraída | Ração/gordura rançosa | **Lesão cerebelar** (Purkinje); dieta |
| **Def. vitamina B1 (tiamina)** (Módulo 4) | **"Stargazing"** (cabeça p/ trás), convulsões | Tiaminase (peixe cru) | Resposta à tiamina |
| **Def. vitamina B2 (riboflavina)** (Módulo 4) | **Dedos curvados** ("curled-toe") | Jovens | Nervos ciático/braquial alargados |
| **Hipocalcemia** (`M02`) | Paralisia/incapacidade de levantar | Poedeira em pico | Responde a cálcio |
| **Botulismo** (`B10`) | **Paralisia flácida ascendente** (pernas→asas→pescoço→3ª pálpebra); "limberneck" | Carcaças/larvas/água; qualquer idade | **Necropsia SEM lesões** + demonstração da **toxina** (soro/conteúdo GI); penas soltas |

## Regras de raciocínio
- **Paralisia assimétrica + nervo aumentado** → **Marek** (idade ≥ 3 sem).
- **Tremor de cabeça em pintinho** → **Encefalomielite** (checar matrizes).
- **Ataxia + lesão cerebelar** → **vitamina E**; **sem lesão + convulsão ativa** → considerar B6/B1.
- **Torcicolo/paralisia sistêmica + alta mortalidade** → **Newcastle/IA** (notificar, laboratório).
- **Paralisia flácida ascendente + necropsia SEM lesões** → **botulismo** (`B10`); procurar
  **fonte** (carcaças/larvas) e demonstrar a **toxina**.
- **Sempre descartar causa nutricional** antes de assumir infecciosa (Módulo 4, arquivo 11).

# Matriz — Sinais Clínicos × Doenças

> Matriz de orientação: parte do **sinal** para as doenças a considerar, com **observação
> distintiva** (não apenas marcação). Fontes = arquivos individuais + Módulo 4. Confirmação sempre
> por laboratório/necropsia.

| Sinal clínico | Doenças a considerar (arquivo) | Observação que ajuda a distinguir |
|---|---|---|
| **Morte súbita, alta mortalidade** | IA (`V01`), Newcastle (`V02`), Cólera (`B06`), Morte súbita/ascite (`M01`) | IA/Newcastle: sistêmico + notificar; Cólera: adultos + focos hepáticos; SDS: ave de dorso, sem agente |
| **Dispneia / respiração de bico aberto** | BI (`V04`), LTI (`V06`), MG (`B04`), ORT (`B07`), aMPV (`V12`), Aspergilose (`F01`), estresse térmico (Módulo 2/5) | LTI: sangue; Aspergilose: sem estertores; MG: crônico; térmico: sem lesão |
| **Muco/sangue na traqueia** | LTI (`V06`) | Traqueíte hemorrágica com cilindros |
| **Cabeça/cara inchada** | Coriza (`B05`), aMPV/SHS (`V12`), Cólera crônica (`B06`) | Coriza: rápida, bacteriana; aMPV: viral + secundária |
| **Torcicolo / paralisia / andar em círculo** | Newcastle (`V02`), Marek (`V05`), EA (`V07`), def. vit. E/B (Módulo 4), hipocalcemia (`M02`) | Marek: assimétrica + nervo; EA: pintinho + tremor; vit.E: cerebelo |
| **Tremor de cabeça (pintinho)** | Encefalomielite Aviária (`V07`) | 1–3 semanas; checar matrizes |
| **Diarreia branca (pintinho)** | Pulorose (`B01`), paratíficas (`B02`) | Onfalite; cultura |
| **Diarreia sanguinolenta** | Coccidiose cecal (`P01`) | Cores cecais |
| **Diarreia amarelo-enxofre** | Histomonose (`P02`) | Fígado em alvo (perus) |
| **Claudicação / artrite** | Reovírus (`V11`), Staph (`B09`), MS (`B04`), Marek (`V05`), discondroplasia/raquitismo (`M02`/Mód. 4) | Pus = bacteriana; sem pus = viral; nervo = Marek |
| **Queda de postura** | EDS (`V08`), BI (`V04`), Newcastle (`V02`), MG/Coriza (`B04/B05`), nutrição/manejo (Módulos 4/5) | EDS: aves sadias + casca ruim; ver DD específico |
| **Casca fina/mole/pálida** | EDS (`V08`), BI (`V04`), def. Ca/P/D/Mn (Módulo 4) | EDS: sorologia HI; nutrição: dieta |
| **Palidez / anemia** | CAV (`V10`), aflatoxina (Módulo 4), ácaro vermelho (`P04`), def. ferro/B12 (Módulo 4) | CAV: medula pálida + timo; ácaro: inspeção noturna |
| **Paralisia assimétrica (perna à frente/atrás)** | Marek (`V05`) | Aumento de nervo periférico |
| **Paralisia flácida ascendente + "limberneck"** | Botulismo (`B10`) | **Necropsia SEM lesões** + toxina no soro/conteúdo GI; fonte = carcaças/larvas; penas soltas |
| **Dedos curvados ("curled-toe")** | Def. riboflavina B2 (Módulo 4) | Nervos ciático/braquial alargados |
| **Tumores viscerais** | Marek (`V05`), Leucose (`V13`), Reticuloendoteliose (`V14`) | Idade/bursa/nervo (ver `V05`) |
| **Lesões de pele (crostas)** | Bouba cutânea (`V09`) | Corpúsculos de Bollinger |
| **Onfalite (pintinho)** | *E. coli* (`B03`), Salmonelas (`B01/B02`), Enterococcus (`B09`) | Higiene de incubatório/casca |

> ⚠️ Nenhum sinal isolado fecha diagnóstico. **Distinguir sinal frequente / possível / inespecífico**
> e correlacionar com **idade, lesões, epidemiologia e laboratório** (regra do plano).

## Fontes
Todos os arquivos de doença do Módulo 9 + Módulos 4 (nutrição) e 5 (manejo).

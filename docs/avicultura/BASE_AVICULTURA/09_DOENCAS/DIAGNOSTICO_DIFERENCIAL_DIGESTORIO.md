# Diagnóstico Diferencial — Síndrome Digestória (Enterites/Diarreia)

> Cruza doenças com sinais entéricos. Fontes = arquivos individuais + Módulo 4 (nutrição/micotoxinas).

| Doença/condição (arquivo) | Pistas | Lesão que distingue | Confirmação |
|---|---|---|---|
| **Coccidiose** (`P01`) | Diarreia (às vezes **sanguinolenta**), aves jovens/recria | **Cores cecais** (E. tenella) ou placas no delgado; **raspado com oocistos** | Necropsia + raspado + OPG |
| **Enterite Necrótica** (`B08`) | **Mortalidade súbita** 2–6 sem, fezes fétidas; frequentemente **pós-coccidiose** | **Membrana diftérica** aderente no jejuno/íleo, intestino friável | Lesão + esfregaço (Gram+) |
| **Histomonose** (`P02`) | Diarreia **amarelo-enxofre**, perus | **Lesões hepáticas em alvo** + cecais | PCR/histopatologia |
| **Newcastle velogênica** (`V02`) | Diarreia verde + respiratório/nervoso | Hemorragia proventricular; botões intestinais | RT-PCR (notificar) |
| **Salmoneloses (jovens)** (`B01/B02`) | Diarreia branca, mortalidade de pintinhos, onfalite | Saco vitelínico não absorvido; nódulos viscerais | Cultura/sorotipagem |
| **Candidíase** (`F02`) | Atraso, papo espessado ("azedo") | **Pseudomembranas brancas no papo** ("toalha turca") | Citologia/cultura |
| **Helmintos** (`P03`) | Queda de desempenho/postura, palidez | Vermes no lúmen (Ascaridia grande); enterite | OPG/necropsia |
| **Micotoxicose** (Módulo 4, `09`) | Baixo desempenho, imunossupressão, lesões variadas | Fígado (aflatoxina), lesões orais (T-2) | Análise de ração |
| **Nutricional/disbiose** (Módulo 4) | Fezes de ração/malabsorção | Inespecífico | Descartar infeccioso; dieta |

## Regras de raciocínio
- **Diarreia sanguinolenta + cecos** → coccidiose (E. tenella); **membrana no delgado** → enterite necrótica (checar coccidiose associada).
- **Fígado em alvo + cecos** (perus) → histomonose.
- **Pintinho com onfalite/diarreia branca** → salmoneloses (cultura).
- **Papo alterado** → candidíase.
- Sempre avaliar **coinfecção** (coccidiose + clostridium) e causas **nutricionais/micotoxinas**.

## Fontes
Arquivos `P01, P02, P03, B01, B02, B08, F02, V02` + Módulo 4 (`09_MICOTOXINAS`, `11_DEFICIENCIAS...`).

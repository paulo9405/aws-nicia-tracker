# Matriz — Órgão × Lesão

> Parte do **órgão** examinado na necropsia para as **lesões** que ele pode apresentar e as doenças
> associadas. Ferramenta de bancada durante a necropsia. Fontes = arquivos 03 + Módulo 9.

| Órgão | Lesões possíveis | Doenças/causas associadas (arquivo) |
|---|---|---|
| **Traqueia** | Hemorragia + cilindros; catarral; caseosa | LTI (`V06`); BI/Newcastle/aMPV; difteria de bouba (`V09`) |
| **Sacos aéreos** | Espessamento/turvação; espuma; caseoso | MG (`B04`), *E. coli* (`B03`), ORT (`B07`), BI (`V04`) |
| **Pulmão** | Nódulos branco-amarelados; congestão | Aspergilose (`F01`); IA (`V01`) |
| **Proventrículo** | Hemorragia de mucosa | Newcastle velogênica (`V02`) |
| **Moela** | Erosão de revestimento | Micotoxinas/def. (Mód. 4); B6 |
| **Intestino delgado** | Membrana diftérica; enterite; placas | Enterite necrótica (`B08`); coccidiose (`P01`) |
| **Cecos** | Cores com sangue; caseoso; lesão | Coccidiose (`P01`); histomonose (`P02`); tiflite |
| **Fígado** | Focos necróticos puntiformes; "em alvo"; gorduroso/hemorrágico; tumor | Cólera (`B06`); histomonose (`P02`); FLHS (`M01`); Leucose/Marek |
| **Baço** | Aumento; necrose | Septicemias; REV (`V14`); Newcastle |
| **Bursa de Fabricius** | Edema/hemorragia→atrofia; tumor nodular | Gumboro (`V03`); Leucose (`V13`) |
| **Timo** | Atrofia | CAV (`V10`); REV (`V14`) |
| **Coração** | Pericardite fibrinosa; dilatação direita; petéquias | Colibacilose (`B03`); ascite (`M01`); cólera/IA |
| **Rim** | Inchado + uratos; gota | BI nefropatogênica (`V04`); desidratação/gota |
| **Ovário/Oviduto** | Folículos hemorrágicos/caseosos; salpingite; peritonite de gema | Pulorose (`B01`); *E. coli*/Gallibacterium (`B03`/`B09`); Newcastle/EDS |
| **Nervos periféricos** | Aumento assimétrico | Marek (`V05`); REV (`V14`) |
| **Encéfalo** | Sem lesão macro (histológica) | Encefalomielite (`V07`); def. vit. E/B (Mód. 4) |
| **Articulações/tendões** | Purulenta; tenossinovite não purulenta | Staph/MS (`B09`/`B04`); reovírus (`V11`) |
| **Ossos** | Cunha cartilaginosa; moles/"beading" | Discondroplasia/raquitismo (`M02`) |
| **Pele** | Crostas/nódulos; dermatite de coxins; celulite | Bouba (`V09`); nutrição/ambiente; *E. coli* (`B03`) |
| **Saco vitelínico (pintinho)** | Não absorvido/infectado (onfalite) | *E. coli*/Salmonela (`B03`/`B01`); má incubação (Mód. 7) |

## Como usar
- Durante a necropsia, ao encontrar uma alteração em um órgão, consultar a linha correspondente para
  **listar as possibilidades** e decidir **qual amostra** coletar (arquivo 04 / Módulo 11).
- Confirmar sempre por **correlação + laboratório** (não fechar pela lesão isolada).

## Fontes
Arquivos `03_LESOES_POR_SISTEMA` + Módulo 9 (doenças, com fontes primárias Merck/MSD/WOAH/PMC).
